## 📌 Overview

**Tres Marias** is a Progressive Web Application (PWA) designed to help small and neighborhood bakeries manage their operations efficiently. It provides a centralized platform for product browsing, pre-ordering, inventory tracking, and hyper-local delivery.

The system aims to bridge the gap between traditional bakery operations and modern digital demands by offering a cost-effective, mobile-first solution.

---

## 🎯 Objectives

* Provide an easy-to-use platform for bakery product browsing and pre-ordering
* Improve inventory management with real-time updates
* Streamline order processing and delivery coordination
* Enhance customer experience through notifications and tracking
* Support small bakery businesses in digital transformation

---

## 🚀 Features

### 👤 User Features

* Browse available bakery products
* Place pre-orders for pickup or delivery
* Real-time order tracking
* Push notifications (order updates, confirmations)
* Mobile-friendly and offline-capable interface

### 🏪 Admin / Bakery Features

* Product management (add, update, remove items)
* Inventory monitoring
* Order management dashboard
* Delivery scheduling and tracking

## 📊 Scope and Limitations

### Scope

* Designed for small, local bakeries
* Supports pre-orders, inventory tracking, and delivery
* Accessible via smartphones, tablets, and desktops

### Limitations

* Requires internet connectivity for full functionality
* Delivery is limited to a defined local area
* No advanced logistics (e.g., route optimization)
* Basic-level security for small-scale deployment
